## SwiftUI Layouts - VStack

SwiftUI has container views as VStack:

```swift
struct VStackExample: View {
   var body: some View {
       VStack {
           Text("Hello")
           Text("World")
           Text("This")
           Text("is")
           Text("Hello")
       }
   }
}
```
